Welcome to Work Hour Tracker

Work Hour Tracker is a simple tool that helps track your daily working hours using an interactive calendar. You can enter start and end times for each workday, and the system automatically calculates the hours worked. Any hours exceeding 8 per day are saved into a time bank.

You can also add hours manually, use saved bank hours, and visualize logged hours by day.

Use the menu to switch between the home page and the actual tracking tool.
